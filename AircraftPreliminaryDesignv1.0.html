Take-off Weight Estimation

$W_{\text{TO}} = \frac{W_{\text{PL}} + W_{C}}{1 - \frac{W_{F}}{W_{\text{TO}}} - \frac{W_{E}}{W_{\text{TO}}}}$

$$W_{\text{TO}}\ :Take - off\ weight$$$$W_{\text{PL}}\ :Payload\ weight$$$$W_{C}\ :Crew\ weight$$$$W_{F}\ :Fuel\ weight$$$$W_{E}\ :Empty\ weight$$

Payload Weight:

$$W_{\text{PL}} = W_{\text{passengers\ }} + \ W_{\text{baggage}}$$$$W_{\text{PL}} = \left( 400 \times 200 \right) + \left( 400 \times 30 \right) = 92000\ lbs \approx 41000\ kg$$

Crew Weight:

$$W_{C} = W_{\text{crew\ }} + \ W_{\text{baggage}}$$$$W_{C} = \left( 10 \times 200 \right) + \left( 10 \times 30 \right) = 2300 lbs \approx 1000\ kg$$

Fuel ratio:

Cruise: (using Breguet Range Equation)

Assumptions:

$\left( \frac{L}{D} \right)_{\max}$= 16

R = 3500 nmi = 6500 km

$$v_{c} = 0.82\ Mach = 900\frac{\text{km}}{\text{hr}}\text{from similar aircraft}$$

This cruise velocity is Carson’s velocity, not velocity as $\left( \frac{L}{D} \right)_{\max}$.

$$v_{\left( \frac{L}{D} \right)_{\max}} = \frac{v_{c}}{1.32} \approx 680\ km/h$$$$\left( \frac{W_{i + 1}}{W_{i}} \right) = e^{\frac{- R \times C \times g}{0.866 \times v_{c} \times \left( \frac{L}{D} \right)_{\max}}}$$

C: specific fuel consumption = 14.1 mg/N.s for cruise for high-bypass turbofan engines (assumption from historical data)

$$\left( \frac{W_{i + 1}}{W_{i}} \right)_{\text{cruise}} = 0.78$$

Loiter:

Considering a loiter time of 30 minutes:

$$\left( \frac{W_{i + 1}}{W_{i}} \right)_{\text{loiter}} = e^{\frac{- E \times C \times g}{\left( \frac{L}{D} \right)_{\max}}} = 0.968$$

C = 17.625 mg/N.s for loiter

Other conditions: (from historical data)

$$\left( \frac{W_{i + 1}}{W_{i}} \right)_{\text{taxi and takeoff}} = 0.98$$$$\left( \frac{W_{i + 1}}{W_{i}} \right)_{\text{climb}} = 0.97$$$$\left( \frac{W_{i + 1}}{W_{i}} \right)_{\text{descent}} = 0.99$$$$\left( \frac{W_{i + 1}}{W_{i}} \right)_{\text{approach and landing}} = 0.997$$

Net weight ratio after landing and before take-off:

$$\left( \frac{W_{\text{AL}}}{W_{\text{TO}}} \right) = 0.78\ \times 0.968 \times 0.98 \times 0.97 \times 0.99 \times 0.97 = 0.705$$

Due to safety considerations, 5% of aircraft weight of fuel must be kept extra.

So,

$$\left( \frac{W_{F}}{W_{\text{TO}}} \right) = 1.05 \times \left( 1 - \frac{W_{\text{AL}}}{W_{\text{TO}}} \right) = 0.309$$

Empty Weight:

Empty Weight fraction from similar aircraft:

Airbus A340: 0.48

Boeing 777-300: 0.525

Airbus A380-841: 0.485

Using this, we have considered empty weight fraction,

$$\left( \frac{W_{E}}{W_{\text{TO}}} \right) = a \times W_{\text{TO}} + b,\ where\ a = \ - 7.754 \times 10^{- 8}\ and\ b = 0.576$$$$We\ get\ this\ value\ to\ be\ 0.5547\ after\ evaluation.$$
